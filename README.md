# WERKHOLZ

Custom solid wood tables from our workshop.

A multilingual Next.js website for a custom woodwork business serving Berlin and Germany.

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Languages

- German (de) — default
- English (en)
- Czech (cs)

Visit `/de`, `/en`, or `/cs` to change languages.

## Project Structure

- `/app` — Next.js pages and layouts
- `/components` — Reusable React components
- `/lib` — Utilities (i18n, API clients, constants)
- `/locales` — Language JSON files
- `/public` — Static assets
